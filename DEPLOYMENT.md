# Web Demo Deployment

This repository is ready for Streamlit Community Cloud deployment.

## Required files already included

- `app.py`
- `requirements.txt`
- `runtime.txt`
- `.streamlit/config.toml`

## Publish steps

1. Push code to GitHub.
2. Create app at <https://share.streamlit.io/>.
3. Select branch and set main file to `app.py`.
4. Deploy.
